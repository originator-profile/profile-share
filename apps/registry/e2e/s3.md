# ファイルアップロードのテスト (minio)

Step1.

[aws-cli](https://github.com/aws/aws-cli)をインストールする。例として、Ubuntu(WSL含む)の場合`apt install awscli`, macOSの場合`brew install awscli`

Step2.

```
$ yarn dev
```

Step3.

```console
$ AWS_ACCESS_KEY_ID=root AWS_SECRET_ACCESS_KEY=kHslkqy4n2hMDvQabcwkOl_NCqdX1M2QEX_XjGe807o aws --endpoint-url http://localhost:19000 s3api create-bucket --bucket test-bucket
{
    "Location": "/test-bucket"
}
$ cat << EOF > test-file
test file
EOF
$ AWS_ACCESS_KEY_ID=root AWS_SECRET_ACCESS_KEY=kHslkqy4n2hMDvQabcwkOl_NCqdX1M2QEX_XjGe807o aws --endpoint-url http://localhost:19000 s3 cp test-file s3://test-bucket/
$ AWS_ACCESS_KEY_ID=root AWS_SECRET_ACCESS_KEY=kHslkqy4n2hMDvQabcwkOl_NCqdX1M2QEX_XjGe807o aws --endpoint-url http://localhost:19000 s3 ls s3://test-bucket/
2023-09-04 13:26:06         10 test-file
```

Step4.

http://localhost:19001/ にブラウザして確認。`Username`として`root`、`Password`として`kHslkqy4n2hMDvQabcwkOl_NCqdX1M2QEX_XjGe807o`を入力しログインすること

# link

グラフィカルなボタンを除く、普通のテキストのリンク

## ExternalLink

外部リンク

```tsx
<ExternalLink href="https://example.org/">Example Domain</ExternalLink>
```

```html
<a
  className="text-blue-600 hover:underline"
  href="https://example.org/"
  target="_blank"
  rel="noreferrer"
>
  Example Domain
</a>
```

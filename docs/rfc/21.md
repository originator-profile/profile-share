# Certificate Category - Editor's Draft

- OP CIP で Certificate を分類しています。
- OP を検証するアプリケーションは、特定のカテゴリーの Certificate の不在など、カテゴリーに基づいた検証失敗のルールを独自に追加することができます (MAY)。
- Certificate を表示するアプリケーションは、分類に従って、 Certificate の表示順序を調整して構いません (MAY)。アプリケーション独自の表示順序の決定方法を採用しても構いません。
- 認証機関は、自身の発行する Certificate の間で、表示順序を指定することができます (MAY)。例えば、 JICDAQ（デジタル広告品質認証機構）は、自身の発行する Certificate の表示順序を[下のリスト](#now)の 1 から 6 のように設定することができます。
- Certificate を表示するアプリケーションは、同じ認証機関による複数の Certificate を表示する際にはこれを守るべきです (SHOULD)。

## Certificate の分類

- 銀行業の免許
- 生命保険会社登録

- など

国際団体による認証

- ISO 認証
  - ISO 27001（情報セキュリティマネジメントシステム）
  - ISO 9001（品質）
  - ISO 14001（環境マネジメント）
- HACCP（Hazard Analysis Critical Control Point：危害要因分析重要管理点）認証 [コーデックス委員会](https://www.fao.org/fao-who-codexalimentarius/about-codex/codex60/en/)策定、自治体、業界団体、民間が認証
- 公正労働協会(FLA)認定 https://www.fairlabor.org/
- など

日本の標準化団体の認証

- プライバシーマーク (JIS Q 15001)

- [JICDAQ](https://www.jicdaq.or.jp/)
- [Trustworthy Accountability Group (TAG)](https://www.tagtoday.net/certifications)

業界団体への参加

- など

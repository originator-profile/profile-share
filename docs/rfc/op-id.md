# Originator Profile Identifier (OP ID)

OP ID は Core Profile 保有組織に与えられる ID です。 URL でなければなりません (MUST)。

[レジストリ](#registry)に登録された形式の OP ID を利用することができます。
登録が受理されるためには OP ID から OP ID 保有組織の公開鍵の集合を得る方法を定める必要があります (MUST)。

## レジストリ {#registry}

- [DNS URI OP ID](./dns-uri-op-id.md)

# 画像の表示

このページでは、本実験で利用する Originator Profile 拡張機能における、画像の表示のされ方について説明します。画像は、OP や DP を発行する際に image プロパティに画像の URL を指定することで表示できます。この画像は、拡張機能を利用するユーザーが、 OP, DP があるページ上で拡張機能を開いた際に見ることができます。

なお、ここで説明する内容は、拡張機能のバージョン v0.0.5 にしたがっており、その後のバージョンアップで変更になる可能性があります。

## 表示領域

「[署名付き広告プロファイル](/terminology/signed-advertisement-profile.md)」に含まれる画像は幅 120px \* 高さ 80px の表示領域にアスペクト比を保持して表示されます。表示領域より小さい画像サイズの場合は、等倍で縦横センタリングします。表示領域より大きい画像サイズの場合は、アスペクト比を保持して縮小表示します。画像フォーマットとしては一般的なブラウザで表示可能なものを指定してください（ img 要素として表示されます）。image プロパティを指定しない場合は画像がないことを示す画像が表示されます。

また、画像は拡張機能画面の左側のアイコンでも使用されます。アイコンは幅 44px \* 高さ 44pxの表示領域に表示されます。画像はアスペクト比を保持して拡大・縮小され、表示領域全体を画像が覆うように表示されます。表示領域に収まらなかった部分は切り取られます。image プロパティを指定しない場合は灰色一色の画像が表示されます。

アスペクト比の等しい大きな画像を使用した場合（画像例 600 \* 400 px）

import image600x400 from "./assets/image-600x400.png";

<img
  src={image600x400}
  alt={"アスペクト比の等しい大きな画像を使用した場合"}
  style={{ maxWidth: 520 }}
/>

アスペクト比の異なる大きな画像を使用した場合（画像例 300 \* 250 px）

import image300x250 from "./assets/image-300x250.png";

<img
  src={image300x250}
  alt={"アスペクト比の異なる大きな画像を使用した場合（画像例 300 * 250 px）"}
  style={{ maxWidth: 520 }}
/>

### 署名付きサイトプロファイルの画像の表示領域

「[署名付きサイトプロファイル](/terminology/signed-site-profile.md)」の場合、画像は幅 80px \* 高さ 45px の表示領域にアスペクト比を保持して表示されます。表示領域より小さい画像サイズの場合は、等倍で縦横センタリングします。表示領域より大きい画像サイズの場合は、アスペクト比を保持して縮小表示します。画像フォーマットとしては一般的なブラウザで表示可能なものを指定してください（ img 要素として表示されます）。

import image800x450 from "./assets/image-800x450.png";

<img
  src={image800x450}
  alt={"アスペクト比の等しい画像を使用した場合"}
  style={{ maxWidth: 520 }}
/>

## 画像へのアクセス許可

画像を正しく表示するにはその画像への外部からのアクセスを許可する必要あります。

画像へのアクセス許可には CORS (Cross-Origin Resource Sharing) の `Access-Control-Allow-Origin: *` への設定変更や HTTP 認証の無効化などが挙げられます。

# ツールバーへの追加

## 対象読者

- 実験参加者

## 前提条件

- [拡張機能のインストール](./experimental-use.md#インストール)

## Google Chrome

1. ブラウザ右上にある「🧩 拡張機能」ボタンをクリック
1. 「[拡張機能を固定](https://developer.chrome.com/docs/extensions/get-started/tutorial/hello-world?hl=ja#pin_the_extension)」: Profile Web Extension の横にある「📌 固定」ボタンをクリック

![利用法の確認1](assets/how_to_use_ext01.png)

## Firefox

1. ブラウザ右上にある「🧩 拡張機能」ボタンをクリック
1. Profile Web Extension の横にある「⚙ 設定」ボタンをクリック
1. 「ツールバーにピン留め」を選択

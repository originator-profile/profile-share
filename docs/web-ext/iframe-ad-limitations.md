# 広告プロファイルのiframe要素からの取得・検証・表示に関する制約事項

## オーバーレイ表示

### マーカーの表示位置は祖先 iframe 要素を元に計算しています

広告プロファイルが設置された広告クリエイティブは、iframe要素をネストした構造で配信されます。

子孫iframe要素や、広告プロファイルの検証時に特定される広告クリエイティブのHTML要素は、クロスオリジンにおけるブラウザの保護機構により、オーバーレイ表示をおこなっている閲覧コンテキストからはDOMが取得できません。

### 祖先 iframe 要素を汚染します

広告プロファイルとiframe要素の対応付けは、広告プロファイルの識別子をデータ属性に格納することでおこなっています。

そのため、拡張機能によってウェブページがウェブページ管理者の意図しない改変を受けます。

また、拡張機能がiframe要素に対して追加したデータ属性の値の真正性は担保されません。

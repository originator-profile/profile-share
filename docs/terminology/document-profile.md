# ドキュメントプロファイル / Document Profile (DP)

コンテンツの基本情報を記載したプロファイル。

DP は対象となるものの種類に応じて最適なスキーマを定義しており、一般的な Web コンテンツを対象とする (狭義の) Document Profile の他に広告を対象とする [Advertisement (Ad) Profile](./advertisement-profile.md) があるがそれらを総称して (広義の) DP と呼ぶ。これらの関係性などが分かり易い用語に将来変更する可能性あり。

コンテンツ作者が対象となるコンテンツの情報を記載して自己署名する。DP に署名をしたデータやファイルは [SDP](./signed-document-profile.md) と呼ぶ。

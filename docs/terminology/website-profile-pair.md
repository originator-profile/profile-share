# ウェブサイトプロファイルペア / Website Profile Pair (Website PP)

[Profile Pair](./profile-pair.md) の一種。 [OP](./originator-profile.md) と、その組織が発行した [SP](./site-profile.md) を 1:1 で組み合わせたペアのデータ。

[SP](./site-profile.md) を使ってサイトの保有者を表明する用途に使われる。サイト保有者は（現行の仕様では）サイトのドメインに `/.well-known/pp.json` のパスで Website Profile Pair を配置し、自身がサイト保有者であることやサイトの情報を真正性のある形でユーザーに提示することができる。

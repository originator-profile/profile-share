# OP, DP レジストリのアクセス制限について

実証実験で使用している [Originator Profile 技術研究組合 (OP CIP)](https://originator-profile.org/) 提供の OP レジストリ、 DP レジストリは**すべてのレジストリ API エンドポイントに Basic 認証を設定しています**。本番運用ではパブリックアクセスにすることを想定している API も含みます。

OP 拡張機能にはこのための認証情報が含まれています。OP レジストリ、 DP レジストリには OP 拡張機能をインストールしたブラウザでアクセスしてください。そのブラウザでのレジストリへのアクセスが自動で認証済みになります。

SDP の登録、削除などの API には実験期間中の追加の認証はありません。これらの API アクセスへのアクセスには引き続き OP 発行時に共有しましたアカウント認証情報または[試験用アカウント](./sample-account.md)をお使いください。

# デプロイグループ

デプロイチームは、OP/CAサーバの運用を目指し、OP技術のデプロイに取り組むグループです。

## 実験目的

- OP 対応をお試し頂くことを通じて OP 技術と仕組みを理解し、各社の実際の CMS やコンテンツ流通の仕組みにおいて支障がないか確認または検討いただきます。
- OP/CAサーバのリファレンス実装をご確認いただき、仕組みや仕様などに関するフィードバックをいただきます。
- 実証実験での検証を目的とした機能のみを実装したリファレンス実装は参考ととしつつ、正式に本番運用するシステムに組み込む際の運用、セキュリティ、課金システムなどまで考慮したOP/CAサーバの設計や実装に着手いただき、来年度以降の正式運用のに向けた準備を始めていただきます。

## 検証内容

- 開発チームが開発したOP/CAサーバのプログラムをご確認いただきます
- 各社のご希望や必要に応じて、先行調査グループと同様に検証サイトを用意して全体の仕組みの検証をしていただきます。
- その他、正式に本番運用するシステムの開発や導入企業・自体対などへの支援に向けて必要となる技術的な議論や仕様へのフィードバックなどを行います。

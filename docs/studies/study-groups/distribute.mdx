# 再配信グループ

再配信グループは、通信社・新聞社等により公開された記事を配信するプラットフォームへのOP対応に取り組むグループです。

## 実験目的

- OP 対応をお試し頂くことを通じて OP 技術と仕組みを理解し、各社の実際の CMS の仕組みにおいて支障がないか確認または検討いただきます。
- 配信元の記事について、その公開、更新、削除などのワークフローの段階で適切に OP (CA) の作成/再配信が出来ることを確認または検討いただきます。

## 検証内容

- ご用意いただいた配信プラットフォーム(本番系か検証系かは任意)において、実際にサイトプロファイルを設置をしていただきます。
- 記事（実際の記事に相当するもの）をご用意いただいた配信プラットフォームを介して配信することを想定し、記事に対してCAを発行いただきます。
- 拡張機能の利用者がサイト全体でOP (CA) が問題なく表示されるかをご確認いただきます。
- OP技術の導入による追加のサーバ負荷や容量への影響が問題のないものであるかを検証いただきます。
- 記事の公開・更新についてのワークフローへの影響が問題のないものかであるかを検証いただきます。
- 実際のシステムへの実装と試験が難しい場合、実際のシステムでこれらの対応をすることが現実的に可能そうか、実際のサンプルデータを元に実際にやり取りするデータのサンプルを手動で用意して机上検討をして頂きます。

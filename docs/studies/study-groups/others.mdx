# その他の参加企業のみなさま

OP/CA対応を自社のサイトに導入し、その仕組みを検証いただくグループです。

## 実験目的

- OP 対応をお試し頂くことを通じて OP 技術と仕組みを理解し、各社の実際の CMS の仕組みにおいて支障がないか確認または検討いただきます。

## 検証内容

- ご用意いただいたサイトにおいて、実際にサイトプロファイルを設置いただきます。
- 実際の記事（実際の記事に相当するもの）に対してCAを発行いただきます。
- 拡張機能の利用者がサイト全体でOP (CA) が問題なく表示されるかをご確認いただきます。
- OP技術の導入による追加のサーバ負荷や容量への影響が問題のないものであるかを検証いただきます。
- 記事の公開・更新についてのワークフローへの影響が問題のないものかであるかを検証いただきます。
- 基本的には先行調査グループの実施内容と同じことを各社の任意で実施頂けますが、開発チームから個別に手順のご案内はありません。ドキュメントサイトなどの情報を元に自主的にお取り組みいただきます。

## お問い合わせ・情報共有

- GitHub/Slack: 現状作成予定はありませんが、実施組織が増える場合、先行調査グループを交えて情報共有や質疑ができるチャンネルの作成を検討します。
- メール: 事務局までご連絡ください。

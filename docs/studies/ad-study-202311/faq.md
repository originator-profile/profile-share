# よくある質問

## メディア側実証実験に参加し全ての CMS 実装を完了していなければ参加できないのでしょうか

いいえ。OP レジストリへの登録がされていることは最低要件になりますが、CMS 側の実装は未完了または保留中であっても、今回の実験中にサイトプロファイルの設置、手動でも良いからメイン記事への Document Profile の作成とリンクの追加が済んでいるページが 1 つ以上あるサイトであれば広告連携実験が可能です。

## XX 月までの実装対応が必要なのでしょうか

いいえ。想定スケジュールはそれに間に合う参加企業がいる限りそれに合わせて進行しますが、間に合わない企業については随時送れてまたは次の実験タイミングに合わせて実施して頂けます。状況と要望に併せて事務局とご調整ください。

## Google Ad Manager を利用するケースだけに対応するのでしょうか

いいえ。初回実験としてはスコープを絞り速やかに実験を進めるため、Google Ad Manager を利用しているケースでの実験を行いますが、それ以外のケースについても参加企業の状況と要望を踏まえて順次実験対象として実験仕様を検討・実施していきます。

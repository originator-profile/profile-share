# 編集履歴

## 2023年8月2日

- CMS の実装ガイドに沿った OP 対応以外についてご検討される実験参加企業を想定し、CLI での OP 対応手順をガイドとして記載しました。
- 内容の変更についてご確認いただけるよう、編集履歴を追加しました。

## 2023年7月26日

DP レジストリ DB への接続が不要な状態で SDP を発行いただけるよう、SDP 発行に使用する CLI を `profile-registry publisher:website` から `profile-registry publisher:sign` へと変更しました。

## 2023年07月21日

WordPress プラグインあるいは CLI のセットアップが不要な方法として、JSON Web Key Generator を使用した鍵の生成方法を追加しました。

## 2023年07月19日

JWK 形式のプライベート鍵に対応したことに合わせて例示を更新しました。

## 2023年07月12日

- WordPress 以外の CMS 連携を実装する場合の説明を追加しました。
- 資格情報の取り扱いに関する説明を追加しました。

## 2023年7月11日

- ご利用いただく SDP 登録 API について、input パラメーターを不要とした `/admin/publisher/{id}/dp/` エンドポイントに変更しました。
- `/admin/publisher/{id}/` エンドポイントを非推奨に変更しました。

## 2023年6月30日

初版

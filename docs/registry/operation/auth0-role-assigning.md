# Auth0 ユーザーの role の変更

## 概要

OP登録サイトで利用する Auth0 テナントの全てのユーザーには初回ログイン時に group ロールが付与されるようになっています。
ユーザーに group 以外のロールを付与したい場合には本ページの手順を実施してください。

## role 一覧

次のロールを付与することができます。

| ロール名  | 説明     |
| :-------- | :------- |
| admin     | 管理者   |
| certifier | 認証機関 |
| group     | 組織     |

## 手順

1. [Auth0 管理画面](https://manage.auth0.com) にログイン
1. 左側のメニューの中の "User Management" をクリック
1. "Users" をクリック
1. ロールを変更したいユーザーをクリック
1. "Roles" タブをクリック
1. 古いロールを削除する場合は roles の一覧の中から削除したいロールをクリックしてゴミ箱のアイコンをクリック
1. 新しいロールを追加する場合は、"Assign Roles" をクリックして、新しいロールを選択して "Assign" をクリック

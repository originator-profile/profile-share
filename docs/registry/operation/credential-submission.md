# 資格情報の提出

Originator Profile（OP）は組織の身元を表明するうえで有用な資格情報を含めることができます。

以下の情報を Originator Profile レジストリ管理者に渡してください。Originator Profile レジストリ管理者が Originator Profile レジストリに資格情報を登録します。

| 項目     | 説明                                                       |
| :------- | :--------------------------------------------------------- |
| 資格名   | 資格の名称です。                                           |
| 画像     | 資格の画像です。認証マークなどを指します。                 |
| 認証機関 | 資格の認証を行った OP レジストリの会員です。               |
| 検証機関 | 認証機関から検証業務を請け負った OP レジストリの会員です。 |
| 発行日時 | 資格の発行日時です。                                       |
| 有効期限 | 資格が失効する日時です。                                   |
